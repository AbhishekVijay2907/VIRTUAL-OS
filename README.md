A browser-based Virtual Operating System that recreates the experience of modern desktop environments such as Windows, macOS, and Linux directly in the browser. The platform features a dynamic desktop interface, window management system, built-in productivity applications, user authentication, file management, and customizable themes powered by a Java Spring Boot backend.

---

## 📖 Overview

Virtual OS Dashboard is a full-stack web application designed to simulate a real desktop operating system experience. Users can interact with a virtual desktop, launch applications, manage files, customize settings, and switch between different operating system themes while maintaining a persistent and personalized workspace.

The project demonstrates modern full-stack development concepts including authentication, RESTful APIs, database integration, state management, security, and interactive user interface design.

## 🚀 Getting Started

### Prerequisites

* Java 17+
* Maven
* MongoDB
* Git

### Clone Repository

```bash
git clone https://github.com/your-username/VIRTUAL-OS.git
cd VIRTUAL-OS
```

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### Frontend Setup

Open the frontend directory and launch:

```bash
frontend/index.html
```

or host it using any local server.
